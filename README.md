# VelogImg
VelogImg는 Velog 글 작성 중 마크다운 이미지 문법(![]())을
HTML <img> 태그로 변환하여 이미지의 크기와 정렬을
쉽고 빠르게 조절할 수 있도록 도와주는 크롬 확장프로그램입니다.

이미지 정렬이나 크기 조절을 위해
매번 HTML 태그를 직접 수정해야 했던 불편함을 줄이고,
클릭 몇 번으로 원하는 이미지 설정을 만들 수 있도록 설계되었습니다.

### 주요 기능
- 이미지 클릭 시 즉시 컨트롤 UI 표시
- 좌 / 중앙 / 우 정렬 지원
- 퍼센트 기반 이미지 크기 조절 (1~100%)
- Velog 전용 최적화

### 사용 방법
1. Velog 글 작성 페이지에서 이미지를 삽입합니다.
2. 또는 좌측 마크다운 기반 이미지 텍스트를 클릭하거나 우측 실제 보이는 이미지를 클릭하면 설정 UI가 표시됩니다.
3. 정렬과 크기를 선택하고 확인 버튼을 누르면 반영됩니다.
